単一ファイルに対して、暗号・復号を行うソフトウェアです。
入出力ファイルは、暗号メッセージ構文(Cryptographic Message Syntax:CMS)
（PKCS#7 ＆ RFC.5652）に準拠しています。

又、ＡＥＳ暗号処理部では、アセンブリ言語にてSIME(SSE2)命令セットを使用して
記述されているため、従来より高速に処理可能です。

《注意》
CUI（コマンドライン・ユーザー・インターフェイス）アプリケーションです。
製品版のVisual Studio.netを買う気が無いので、GUIは今のところ考えてません。

暗号のみ対応です。電子署名（Signed-data Type）等には対応していません。



■対応アルゴリズム
━━━━━━━━━━━━
《暗号関数》
・DES-CBC		…　※政府承認暗号から削除されました。
・DES-3EDE-CBC		…　※推奨されません。
・AES-128-CBC
・AES-192-CBC
・AES-256-CBC

《鍵導出関数》		（※パスワード文字列から、暗号鍵を計算する関数です。）
・PBKDF2

《鍵付きハッシュ関数》	（※"PBKDF2"で使う、鍵つきハッシュ関数です。）
・HMAC-SHA-1
・HMAC-SHA-224
・HMAC-SHA-256

《疑似乱数》		（※セッション鍵や、初期化ベクタIVに用います。）
・Mersenne Twister ＋ SHA-256

	【2.00に向けての、※α版みたいなものです。】
	・ソフトの名称を、"AES"から"CMS"に変更した。
	・PKCS#7のEnveloped-Data Typeに対応しました。
	　対応する鍵管理手法は、鍵導出（パスワード）のみ対応しています。
	　（秘密鍵と公開鍵証明書を用いる鍵配送には対応していません。）
	　　・鍵導出関数：PBKDF2 ＋ HMAC-SHA-256
	　　・鍵ラップ　：PWRI-KEK ＋ コンテンツ用暗号と同じ暗号アルゴリズム
	・デフォルトで、Enveloped-Data Typeを処理するようにした。
	・コンテンツ用で、DES、3-DESに対応した。
	　（が、古い暗号ですので、使用はお勧めしません。）
