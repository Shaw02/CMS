CMS対応　暗号／復号ユーティリティ

アルゴリズム（AlgorithmIdentifier）

各アルゴリズムのoidと、そのパラメータが格納されます。
詳細は"RFC.3370"を参照してください。

AlgorithmIdentifier ::= SEQUENCE  {
  algorithm OBJECT IDENTIFIER,
  parameters ANY DEFINED BY algorithm OPTIONAL  }
AlgorithmIdentifierについて
フィールド名 内容
algorithm アルゴリズムを示すoidが格納される。
parameters アルゴリズムに付随するパラメータを指定する。

ダイジェストアルゴリズム（DigestAlgorithmIdentifier）

コンテンツのダイジェスト値（バイト列）を求めるためのアルゴリズムです。
SHA-1等。 

DigestAlgorithmIdentifier ::= AlgorithmIdentifier

鍵管理アルゴリズム（KeyEncryptionAlgorithmIdentifier）

コンテンツ用暗号鍵(CEK)を暗号化するためのアルゴリズムです。
パスワードによる暗号化など。 

KeyEncryptionAlgorithmIdentifier ::= AlgorithmIdentifier

コンテンツ暗号アルゴリズム（ContentEncryptionAlgorithmIdentifier）

コンテンツを暗号・復号するためのアルゴリズムです。
DES, AES等。 

ContentEncryptionAlgorithmIdentifier ::= AlgorithmIdentifier

メッセージ認証符号(MAC)アルゴリズム

コンテンツに改ざんが無いか確認するためのアルゴリズムです。
その他、鍵導出関数PBKDF2にも用いられ、本ソフトではこの用途にて使用しています。
HMAC-SHA-1等。 

MessageAuthenticationCodeAlgorithm ::= AlgorithmIdentifier

鍵導出アルゴリズム（KeyDerivationAlgorithmIdentifier）

パスワードによって鍵暗号用の暗号鍵(KEK)を求めるためのアルゴリズムです。
PBKDF2等があります。

KeyEncryptionAlgorithmIdentifier ::= AlgorithmIdentifier
