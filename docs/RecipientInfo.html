CMS対応　暗号／復号ユーティリティ

受取人情報 （RecipientInfo）

鍵管理の手法によって、それぞれことなる構造のデータが格納されます。
本ソフトでは、鍵導出（パスワード）による暗号のみに対応しています。

RecipientInfo ::= CHOICE {
  ktri KeyTransRecipientInfo,
  kari [1] KeyAgreeRecipientInfo,
  kekri [2] KEKRecipientInfo>,
  pwri [3] PasswordRecipientinfo,
  ori [4] OtherRecipientInfo }

EncryptedKey ::= OCTET STRING
PasswordRecipientinfoについて
フィールド名 内容
KeyTransRecipientInfo 鍵配送　（対称鍵暗号アルゴリズムを用いた鍵管理手法です。）
KeyAgreeRecipientInfo 鍵合意　（鍵合意アルゴリズムを用いた鍵管理手法です。）
KEKRecipientInfo 共通鍵暗号化鍵　（共通鍵暗号アルゴリズム（暗号鍵は事前に共有化）を用いる鍵管理手法です。）
PasswordRecipientinfo 鍵導出　（パスワード等を用いる鍵管理手法です。）
OtherRecipientInfo その他
