暗号化データ(Encrypted-data)タイプ

暗号化されたコンテンツのみを格納するフォーマットです。
暗号鍵の管理はされないため、そのままでは封筒化データよりはセキュリティーに劣ります。
EnvelopedDataは以下のASN.1による記述で定義されます。

[ASN.1] EncryptedData (oid = 1.2.840.113549.1.7.6)
EncryptedData ::= SEQUENCE {
  version CMSVersion,
  encryptedContentInfo EncryptedContentInfo,
  unprotectedAttrs [1] IMPLICIT UnprotectedAttributes OPTIONAL }

UnprotectedAttributes ::= SET SIZE (1..MAX) OF Attribute
EncryptedDataについて
フィールド名 内容
version 2 : unprotectedAttrsフィールドが存在する場合。
0 : unprotectedAttrsフィールドが存在しない場合。
EncryptedContentInfo 暗号化されたコンテンツとその情報。
unprotectedAttrs 暗号化されない属性の集まり。
